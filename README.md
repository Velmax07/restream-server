# IPTV Restream Server

Servidor proxy de re-stream para IPTV. Permite múltiplos usuários assistirem simultaneamente usando a mesma conta IPTV.

## Como usar

Após deploy no Railway, use a URL do servidor assim no seu HTML:

```
https://SEU-SERVIDOR.railway.app/proxy?url=SUA_URL_M3U8
```

## Deploy no Railway

1. Faça upload deste repositório no GitHub
2. Conecte ao Railway
3. Clique em Deploy
4. Pronto!
